这套代码旨在构建一个基于机器学习的大学生心理健康风险预测系统，并生成学术论文风格的统计图表。核心流程为：加载预处理后的数据（processed_data.pkl），剔除两个可能造成标签泄露的特征，分别训练随机森林、SVM 和逻辑回归三个模型（超参数已调优），然后在测试集上计算准确率、精确率、召回率、F1、AUC等指标，并输出随机森林的特征重要性排序。同时，代码通过5折交叉验证评估随机森林的稳定性，并依据训练集交叉验证概率自动搜寻最优决策阈值，为实际部署提供灵活依据。此外，项目还包含四个独立的绘图脚本，将性能对比、交叉验证结果、特征重要性和风险分层比例分别可视化为高分辨率图片，便于直接用于论文或报告。整套代码注重可复现性（固定随机种子）、中文显示（自动加载同目录字体）和可解释性（颜色编码与累计贡献标注），并预留了风险分层策略说明，以辅助“AI自动处理 vs. 人工介入”的决策。

三类机器学习模型性能与特征筛选实验结果(1).py
这是整个项目的核心训练与评估脚本。它首先从 processed_data.pkl 中读取训练集、测试集和特征名称，并移除 Crisis_Severity_Index 和 Mental_Health_Composite 两个直接反映危机严重程度的特征，以防止数据泄露。随后，使用已调优的超参数（随机森林800棵树、SVM的C=0.4、逻辑回归的C=0.3）分别在干净数据上训练三个模型，并在测试集上计算多种分类指标，同时输出随机森林的前10个重要特征及其分数。为了验证模型的泛化能力，脚本还进行了5折分层交叉验证，计算F1的均值和标准差。最后，它利用训练集上的交叉验证预测概率，在0.1~0.9范围内搜索使F1最大的决策阈值，从而优化实际应用中的分类边界。该脚本的终端输出是所有后续图表的数据来源。

三类机器学习模型各评价指标对比柱状图.py
该脚本用于生成三个模型在准确率、精确率、召回率和F1分数上的分组柱状图。它直接使用上一个脚本计算出的具体数值（硬编码在代码中），以马卡龙配色绘制四组并列柱形，并特意将SVM的召回率柱条标为红色，以突出其在该指标上的劣势。图表标题、轴标签和图例均采用中文字体，并且自动加载同目录下的 .ttf/.otf 文件保证中文正常显示。最终输出高分辨率的 figure1_performance_bars.png，直观对比各模型在不同评价维度上的表现。

随机森林5折交叉验证F1分数箱线图.py
此脚本专注于展示随机森林在5折交叉验证中的稳定性。它根据报告中给出的F1均值（0.8544）和标准差（0.0426），反推出五个折叠的F1值（近似服从正态分布），从而绘制箱线图，并叠加散点展示每个折叠的具体分数。箱线图的中位数、四分位距以及散点的分布，能够清晰反映模型在不同数据子集上的波动情况。图上还标注了均值和标准差的数值，便于读者快速评估模型的稳健性。最终输出 figure2_cv_boxplot.png，为验证随机森林的泛化能力提供可视化证据。

模型前10项特征重要性水平条形图.py
该脚本将随机森林输出的特征重要性分数进行可视化，绘制水平条形图，仅显示前10个最重要的特征。特征名称已翻译为中文（如“抑郁评分”“焦虑评分”等），条形按重要性降序排列。颜色编码策略为：前两名（抑郁和焦虑）使用深红色以强调其主导作用，第3~4名（交互项）使用马卡龙杏色，其余则为灰色。同时，在每个条形的右侧标注了累计贡献百分比，直观体现前几个特征对模型预测能力的累积解释程度。图表包含图例和网格线，输出为 figure3_feature_importance.png，有助于解释模型的内在决策依据。

样本风险分层占比环形分布图.py
此脚本用于呈现预测风险等级在整体样本中的分布情况。它将预测概率划分为低风险（<0.1）、中风险（0.1~0.8）和高风险（≥0.8），并以环形图（圆环图）展示各层的比例。代码中当前使用的是演示比例（10%、80%、10%），用户需要根据实际数据替换 risk_sizes 列表。环形图的颜色采用马卡龙风格，并添加了策略说明文字，强调中风险由AI自动处理、高风险需人工介入。该脚本输出 figure5_risk_pie.png，为分层干预策略提供直观的占比参考，便于业务决策。
